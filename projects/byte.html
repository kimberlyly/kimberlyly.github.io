Byte Meet-ups that actually happen!

Byte is a mobile-first web app designed and developed for COGS 187A.

The app aims to solve the problem of friends making plans to go out, but never actually following through due to scheduling conflicts or just forgetfulness. With the simple interface, sending an in-byte is as easy as choosing a friend, then selecting a time and location. If you can't make up your mind, then Byte will randomly choose a location for you! Your friend will have multiple scheduling options, so you can be guaranteed that at least one time works for the both of you. Byte discourages "flaking" by gamifying the experience; the more times you follow through, the more points you get!

Project Details

  • My role on this project was the lead developer.
  • I set up the code framework for the project and delegated coding tasks.
  • I helped in the initial stages of our project with storyboarding, user interviews, and wireframing.
